# Closied Realtor-Integrated Frontend

This React app includes:
- Geolocation prompt and ZIP search
- Realtor (RapidAPI) integration (uses REACT_APP_REALTOR_RAPIDAPI_KEY)
- Logged-in dashboard with sidebar: Profile, Saved, Workflows, Documents

Environment variables (Vercel):
- REACT_APP_API_BASE=https://your-backend-url
- REACT_APP_REALTOR_RAPIDAPI_KEY=your_rapidapi_key
- REACT_APP_REALTOR_RAPIDAPI_HOST=realtor.p.rapidapi.com

Run locally:
```
npm install
npm start
```
